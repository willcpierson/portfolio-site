Live Projects

Local Locator

Frontend, Vanilla JavaScript

A frontend only application that helps users locate and find information on in-person video game tournaments. The user is able to search any video game available on the tournament site start.gg, and Local Locator pulls the tournament information from said site.

Kelp

Yelp Clone: Full Stack, Ruby, Rails, React, Redux

Allows the user to create an account and log in, which lets the user leave reviews for sea related businesses. The user can search for any sea related businesses in NYC, favorite said businesses for easier access on their profile page, and create, update, and delete reviews.

WaveSense

MERN Full Stack, JavaScript, React, Redux, MongoDB

This application lets the user create an account, and when they log in they are able to add songs, create playlists and add to the playlist from pre-existing songs. The main feature has a 3D visualization of the music as it plays.
